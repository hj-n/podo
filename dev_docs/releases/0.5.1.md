# Podo 0.5.1

Podo 0.5.1 completes the public installation documentation and improves interrupted product-update diagnosis.

## Changes

- README now exposes the verified anonymous GitHub `install.sh` command.
- Task startup and `podo doctor` report unfinished product update journals separately from Context transactions.
- Public install, exact-version update and rollback guidance is aligned with the released command surface.

## Compatibility

- Product: 0.5.1
- Workspace format: 1
- No Workspace migration is required from 0.5.0.

After updating, start a new Codex task and review `.codex/hooks.json` before trusting a changed hook.
