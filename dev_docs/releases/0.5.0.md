# Podo 0.5.0

Podo 0.5.0 introduces public GitHub Release installation and safe product updates for Workspace format 1.

## Highlights

- Reproducible, product-only Release archive with published SHA-256
- Standalone `install.sh` and package installer
- `podo update` for the latest stable version
- `podo update --version <version>` for exact compatible update or rollback
- Three-root product transaction covering `AGENTS.md`, `.codex` and `.podo`
- Product drift, checksum, archive path, Workspace compatibility and unfinished Context preflight checks
- Automatic rollback to the previous product after handled apply or validation failure

## User Data

Normal installation and update do not overwrite `WORKSPACE_VERSION`, `.podo-work`, `.podo-backups`, `user_config.md`, `events`, `deltas` or `state`.

## After Updating

Start a new Codex task so the updated operating policy is loaded. Review `.codex/hooks.json` before trusting a new or changed project hook.

## Compatibility

- Product: 0.5.0
- Workspace format: 1
- Supported transcript runtime: `codex-cli 0.144.0-alpha.4`
