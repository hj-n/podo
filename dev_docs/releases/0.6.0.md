# Podo 0.6.0 Candidate

Podo 0.6.0은 별도 승인 Workspace migration과 full rollback infrastructure를 개발하는 candidate다.

## Candidate Features

- Verified target Release의 ordered migration graph와 exact impact plan
- Exact plan approval, stale evidence rejection과 serialized apply
- 이전 제품과 영향받는 사용자 data의 persistent backup
- Staged migration, declared-change enforcement와 journaled product/data apply
- 별도 승인 full rollback과 rollback-start safety backup
- Unfinished migration doctor/startup diagnosis

## Compatibility

- Product: 0.6.0 candidate
- Workspace format: 1
- 실제 Workspace format 2 migration은 포함하지 않는다.
- Public GitHub Release로 publish되지 않았다.
- Migration entrypoint는 verified package code지만 별도 OS sandbox에서 실행되지는 않는다. Engine은 staged Workspace와 declared-change 검증으로 적용 범위를 제한한다.
