# Podo 0.5.2

Podo 0.5.2 fixes public Release downloads on macOS Python installations whose embedded CA store cannot verify GitHub.

## Fix

- `podo update` now uses the documented system `curl` prerequisite for GitHub API and Release asset downloads.
- Checksum, external/internal Release identity and safe extraction behavior are unchanged.
- Download failure still occurs before any product replacement.

## Compatibility

- Product: 0.5.2
- Workspace format: 1
- No Workspace migration is required from 0.5.0 or 0.5.1.

After updating, start a new Codex task and review `.codex/hooks.json` before trusting a changed hook.
