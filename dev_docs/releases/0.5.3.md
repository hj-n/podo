# Podo 0.5.3

Podo 0.5.3 hardens public Release asset origin validation.

## Change

- GitHub API asset URLs must remain under the selected `hj-n/podo` tag's public Release download path.
- Checksum, metadata identity, safe extraction and transactional update behavior are unchanged.

## Compatibility

- Product: 0.5.3
- Workspace format: 1
- No Workspace migration is required from Podo 0.5.x.

After updating, start a new Codex task and review `.codex/hooks.json` before trusting a changed hook.
